# Support Center

The iNZight Support Center is a place to find answers to common questions and solve common problems. Please browse the following pages or use the search bar. If you cannot find the answer, you may use the contact form to send us a message. Please not that we have limited resources and no dedicated support staff, but we will reply as soon as we can.

:::tip Discord Server

We are pleased to announce our new Discord community server! All users are encouraged to use this public space to find answers, help others, and provide suggestions and feedback for iNZight.

:::
