---
title: Delete
description: Delete variables from a dataset
---

Delete variables (columns) from the dataset. Select one or more variable names from the list — hold **Shift** to select a range or **Ctrl** to select multiple individual variables — then click **Delete** to remove them.

:::warning
Deleted variables cannot be recovered except by restoring the original dataset. Use **Dataset** > **Restore original dataset** to undo all changes since the data was loaded.
:::
