title: Reorder levels
description: Reorder levels of a categorical variable.
---

import { Desktop, Lite } from "@site/src/components/VersionToggle/inzVersion";

By default, the levels of a categorical variable are displayed in alpha-numeric order. Use the **Reorder Levels** feature to change from the default order of display to something more natural.

For example, `{"adolescent", "adult", "child", "elder"}` to `{"child", "adolescent", "adult", "elder"}`.

Go to **<Lite>Manipulate </Lite>Variables** > **Categorical Variables** > **Reorder levels**.
