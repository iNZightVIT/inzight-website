## Supported Formats

iNZight can export data to:

- **CSV** (.csv) - Comma-separated values, widely compatible
- **Tab-delimited** (.txt) - Tab-separated values
- **R data** (.rds) - R's native format, preserves data types
- **Excel** (.xlsx) - Microsoft Excel format
- **Stata** (.dta) - Stata data format
- **SPSS** (.sav) - SPSS data format
- **SAS** (.sas7bdat) - SAS data format

## Tips

- Use CSV for maximum compatibility with other software
- Use RDS to preserve factor levels and other R-specific attributes
- When exporting to Excel, be aware that some data types may be converted
