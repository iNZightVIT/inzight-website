The menubar is displayed at the top of the iNZight window, and allows you to things like import data, apply transformations to your data, manipulate variables, and many other things. More advanced features are often placed in here to keep the rest of the interface as simple as possible.

<Figure src={menuImg} caption="The Menu bar" width="400" />

## File

The **File** menu is where you interact with files on your computer (either by importing or exporting data), as well as access example data sets and manage your preferences.

import fileMenu from "./file_menu.png";

<Figure src={fileMenu} caption="The File menu" width="auto" />

- [**Import data**](/docs/reference/import-export/import-data) from a file on your computer, or a remote URL.
- [**Export data**](/docs/reference/import-export/export-data) to a file on your computer.


- **Clipboard**
  - [Paste from](/docs/reference/import-export/clipboard) the clipboard (e.g., copy from Excel / Google Sheets)
  - [Copy to](/docs/reference/import-export/clipboard#copy-to) the clipboard and paste into e.g., Excel or Google Sheets


- [**Example data**](/docs/reference/import-export/example-data) lets your choose from a list of example data sets.



- [**Preferences**](/docs/reference/preferences) lets you change some of the settings in iNZight.
- **Reload iNZight** can be useful if you run into random errors.
- **Exit** closes iNZight.

Click the links above to go to the relevant reference page.

## (Manipulate) Variables

A collection of variable manipulation methods to allow you to convert variables to new types, create new variables, and rename existing ones.

:::note

Called **Manipulate Variables** in iNZight Lite.

:::

import variablesMenu from "./variables_menu.png";

<Figure src={variablesMenu} caption="The Variables menu" width={"auto"} />

- **Convert to Categorical** to convert a numeric variable to categorical.
- **Categorical Variables** to manipulate categorical variables.
  - [Reorder levels](/docs/reference/categorical-variables/reorder-levels) to change the order of the levels of a categorical variable.
  - [Collapse levels](/docs/reference/categorical-variables/collapse-levels) to combine levels of a categorical variable.
  - [Rename levels](/docs/reference/categorical-variables/rename-levels) to rename levels of a categorical variable.
  - [Combine categorical variables](/docs/reference/categorical-variables/combine-categorical) to combine two categorical variables into one.
- [**Numeric Variables**](/docs/category/numeric-variables) to manipulate numeric variables.
  - [Transform](/docs/reference/numeric-variables/transform) to apply a transformation to a numeric variable.
  - [Standardise](/docs/reference/numeric-variables/standardise) to standardise a numeric variable.
  - [Form class intervals](/docs/reference/numeric-variables/form-class-intervals) to convert a numeric variable to categorical.
  - [Rank numeric variables](/docs/reference/numeric-variables/rank) to rank the values of a numeric variable.
  - **Convert to categorical (multiple)** to convert multiple numeric variables to categorical.
- [**Dates and Times**](/docs/category/dates-and-times)
  - [Convert to](/docs/reference/dates-and-times/convert-to) to convert a variable to a date or time.
  - [Extract from](/docs/reference/dates-and-times/extract-from) to extract information from a date or time variable.
  - [Aggregate to](/docs/reference/dates-and-times/aggregate-to) to aggregate a date or time variable to a higher level.
- [**Rename variables**](/docs/reference/data/rename) to rename variables.
- [**Create new variables**](/docs/reference/data/create) to create new variables from existing ones.
- [**Missing to categorical**](/docs/reference/data/missing-to-categorical) to make missing values explicit.
- [**Delete variables**](/docs/reference/data/delete) to delete variables.

## Advanced

The Advanced menu provides a suite of extra features and modules for more specialised analyses:

- **Quick Explore** - Tools for quickly exploring your data, including missing value analysis, all 1-variable plots/summaries, 2-variable exploration, and pairs plots.
- **3D Plot** - Interactive 3D scatter plots.
- **Time Series** - Time series decomposition, seasonal plots, and forecasting.
- **Model Fitting** - Linear regression, generalized linear models, and model diagnostics.
- **Multiple Response** - Analysis of multiple response survey questions.
- **Maps** - Coordinate maps and regional/choropleth maps.

Documentation for these modules is coming soon.
