# Install iNZight Desktop Application

iNZight is available to download on **Windows** and **Linux**.

Windows users can simply download the installer below and follow the installation instructions.

:::tip iNZight Lite works in any browser

Mac and tablet users can use iNZight Lite, which is available [here](https://lite.docker.stat.auckland.ac.nz).

:::

### Installation Instructions

Open the installer and follow the instructions. You can choose the installation directory, but we suggest leaving this as default (Documents/iNZightVIT).

After the installation has completed, you can launch iNZight from

- the desktop shortcut
- the start menu shortcut
- the installation directory (default Documents/iNZightVIT)

See [Start iNZight](/docs/getting-started/loading-inzight) for details.

### Uninstall

Simply run the Uninstaller found in the installation directory (default Documents/iNZightVIT).

## Linux

Installation on Linux requires that you install the necessary dependencies first. We recommended using the [GitHub](https://github.com/iNZightVIT/linux) install script, which includes instructions for various distributions.
