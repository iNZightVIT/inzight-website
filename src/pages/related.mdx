# Related Projects

Here are several projects related to iNZight, or supervised by Professor Chris Wild.

---

## VIT (Visual Inference Tools)

<a href="https://www.stat.auckland.ac.nz/~wild/VIT/index.html" target="_blank" rel="noopener noreferrer">
  Visit VIT →
</a>

VIT is a companion program to iNZight designed to help students understand statistical inference concepts through visual animations. It demonstrates concepts like:

- Sampling variation
- Bootstrap confidence intervals
- Randomisation tests
- Permutation tests
